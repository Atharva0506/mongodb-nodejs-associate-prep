# MongoDB Node.js Associate Prep

Short, exam-focused notes and practice material for MongoDB Associate Developer (Node.js).

## Purpose
- This repository is for educational and self-study use only.
- It summarizes concepts seen in official MongoDB practice questions.

## What is inside
- `mongodb-notes/`: Topic-wise notes (CRUD, aggregation, indexes, schema design, etc.)
- `offical-question.md`: Practice questions collected from the official MongoDB learning source.
